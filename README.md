# ClearOps

**Operational clarity, when it matters.**

ClearOps is an operational visibility platform that helps teams identify
exceptions, track follow-ups, and act before customers are impacted.

It turns raw order, shipment, and tracking data into:
- Clear exception queues
- Supplier accountability
- Proactive customer communications
- Historical trends across runs

---

## What ClearOps Is (and Isnâ€™t)

**ClearOps is:**
- An operations and exception management hub
- Built for messy, real-world CSV exports
- Designed for supply chain, ops, CX, and finance teams
- Safe to explore instantly using Demo Mode

- Locked to dropshipping
- Locked to a single platform

---

## Core Capabilities

- **Ops Triage**
  - Automatically surface Critical and High-urgency issues
  - Focus attention where it matters first

- **Exceptions Queue**
  - Filter by supplier, country, urgency, and issue type
  - Export actionable CSVs

- **Supplier Follow-ups**
  - Track contact status and escalation
  - Auto-fill emails from Supplier Directory (CRM)

- **Customer Communications**
  - Identify customer-impacting issues
  - Generate email-first outreach

- **Scorecards & Trends**
  - Supplier performance scorecards
  - Historical trends from saved runs

- **Demo Mode (Sticky)**
  - Runs with zero uploads
  - Demo edits persist
  - Fully forkable into real workspaces

---

## Getting Started

1. Launch the app
2. Turn on **ClearOps Demo (Sticky)** to explore instantly  
   **or**
3. Upload:
   - Orders CSV
   - Shipments CSV
   - (Optional) Tracking CSV
4. Review Ops Triage and Exceptions
5. Take action and save runs to build history
